# Chart implementation for Synths

LineChart extends Chart.js/LineChart

## Props:

    - labels: Array of ?
    - chartData: Array of objects

## Example:

```
labels: [12, 25, 1, 5, 20, 3],
chartData: [
  {
    backgroundColor: '#FFFFFF',
    borderWidth: 3,
    tension: 0.4,
    fill: false,
    borderColor: '#FFFFFF',
    data: [
      12,
      25,
      1,
      5,
      20,
      3
    ]
  }
]
```
